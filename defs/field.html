A field is a set \(\mathbf F\) containing two special elements (called 0 and 1), as well as two operations, \[+:\mathbf F^2\to \mathbf F\qquad\text{and}\qquad\times:\mathbf F^2\to \mathbf F\] called addition and multiplication, usually written \[+(a, b) := a + b\qquad\text{and}\qquad\times\!(a, b) := ab\] satisfying the following axioms.

  1. Addition is associative: \(\forall a, b, c\in \mathbf F\) we have \(a + (b + c) = (a + b) + c\).
  2. The element 0 is an additive identity: \(\forall a\in \mathbf F\), we have \(a + 0 = a = 0 + a\).
  3. Additive inverses exist: \(\forall a\in \mathbf F\), we have \(a + b = 0 = b + a\) for some \(b\in \mathbf F\).
  4. Addition is commutative: \(\forall a, b\in \mathbf F\) we have \(a + b = b + a\).

  5. Multiplication is associative: \(\forall a, b, c\in \mathbf F\) we have \(a(bc) = (ab)c\).
  6. The element 1 is a multiplicative identity: \(\forall a\in \mathbf F\), we have \(a1 = a = 1a\).
  7. Multiplicative inverses exist for nonzero elements: \(\forall a\in \mathbf F\setminus\{0\}\), we have \(ab = 1 = ba\) for some \(b\in \mathbf F\).
  8. Multiplication is commutative: \(\forall a, b\in \mathbf F\) we have \(ab = ba\).

  9. Multiplication distributes over addition: \(\forall a, b\in \mathbf F\) we have \(a(b + c) = ab + ac\).
  10. The additive identity is distinct from the multiplicative identity: \(0\ne 1\).

Elements of \(\mathbf F\) are called scalars.
