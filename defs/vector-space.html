A vector space (over a field \(\mathbf F\)) is a set \(V\) with:

  • a special element 0 (called the origin),
  • an operation \(+:V^2\to V\) given by \((a, b)\mapsto a + b\) (called vector addition),
  • an action \(\mathbf F\times V\to V\) given by \((\lambda, v)\mapsto \lambda v\) (called scalar multiplication),
satisfying the following axioms:

  1. Vector addition is associative: \(\forall a, b, c\in V\) we have \(a + (b + c) = (a + b) + c\).
  2. The origin is an additive identity: \(\forall a\in V\), we have \(a + 0 = a = 0 + a\).
  3. Additive inverses exist: \(\forall a\in V\), we have \(a + b = 0 = b + a\) for some \(b\in F\).
  4. Vector addition is commutative: \(\forall a, b\in V\) we have \(a + b = b + a\).

  5. We have \(1v = v\) for all \(v\in V\).
  6. We have \(\alpha(\beta v) = (\alpha\beta)v\) for all \(\alpha,\beta\in\mathbf F\) and \(v\in V\).

  7. Scalar multiplication distributes over vector addition: for all \(\alpha,\beta\in\mathbf F\) and \(u, v\in V\) we have \[\alpha(u + v) = \alpha u + \alpha v\qquad\text{and}\qquad (\alpha + \beta)v = \alpha v + \beta v.\]

Elements of \(V\) are called vectors, whereas elements of \(\mathbf F\) are called scalars.
