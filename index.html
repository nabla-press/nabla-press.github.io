  1. In linear algebra there are vectors and scalars.
    • A vector is an element of a vector space over a given field,
    • whereas a scalar is an element of the field itself.
  2. The set \(\{0\}\) is a vector space, known as the trivial space.
  3. Any vector space \(V\) has a set of subspaces.
  4. Any nonempty subset \(X\) of a vector space \(V\) generates a subspace of \(V\), called the span of \(X\).
  5. Every vector space has a basis, and hence a dimension.
  6. Between any two vector spaces \(V\) and \(W\), there is a set of linear maps, denoted \(\Hom(V, W)\).
    • This forms a vector space.
