vector spaces

(back to top)

  1. In linear algebra there are vectors and scalars.
    • A vector is an element of a vector space over a given field,
    • whereas a scalar is an element of the field itself.
  2. The set \(\{0\}\) is a vector space, known as the trivial space.
  3. Any vector space \(V\) has a set of subspaces.
    • One can form direct sums and quotients of subspaces.
  4. Any nonempty subset \(X\) of a vector space \(V\) generates a subspace of \(V\), called the span of \(X\).
  5. Every vector space has a basis, and hence a dimension.
  6. Between any two vector spaces \(V\) and \(W\), there is a set of linear maps, denoted \(\Hom(V, W)\).
    • This forms a vector space.
    • If \(V = W\), we denote \(\Hom(V, W)\) by \(\End(V)\).
    • The set of automorphisms of \(V\) (bijective linear maps from \(V\) to \(V\)) is denoted \(\Aut(V)\).
    • If \(W = \mathbf F\), we denote \(\Hom(V, W)\) by \(V^*\) and call this space the dual space of \(V\).
  7. Every linear map has a kernel and an image.
  8. The universal property of subspaces is dual to the universal property of quotient spaces.
  9. The rank-nullity theorem is a corollary of canonical decomposition (also known as the first isomorphism theorem), which is itself a corollary of the universal property of quotient spaces.
  10. Bilinear maps on the Cartesian product \(V\times W\) are the same thing as linear maps on the tensor product \(V\otimes W\).
  11. The tensor algebra \(T^\bullet(V)\) is: \[T^\bullet(V) := \bigoplus_{k\ge 0}T^k(V)\] where \(T^0(V) = \mathbf F,\; T^1(V) = V,\; T^2(V) = V\otimes V,\; T^3(V) = V\otimes V\otimes V,\;\dots\)
  12. The alternating algebra \(\Alt^\bullet(V)\) (where \(V\) has dimension \(n\)) is \[\Alt^\bullet(V) := \bigoplus_{k= 0}^n\Alt^k(V)\] where \(\Alt^0(V) = \mathbf F,\;\Alt^1(V) = V,\; \Alt^k(V) = \{x_1\wedge\cdots\wedge x_k : x_i\in V\}\). Here, \(\wedge\) is the wedge product.
  13. The map \(T\in\End(V)\) induces a linear map \[\Lambda^nT\in\End(\Alt^n(V))\qquad\text{given by}\qquad v_1\wedge\dots\wedge v_n\mapsto Tv_1\wedge\dots\wedge Tv_n.\] But since \(\Alt^n(V)\) is 1 dimensional, this map must be multiplication by some scalar, i.e. we have \[\Lambda^nT(v_1\wedge\dots\wedge v_n) = D(v_1\wedge\dots\wedge v_n)\] for some \(D\in\mathbf F\). We define the determinant of \(T\) (denoted \(\det T\)) to be this scalar \(D\).
  14. There is a natural isomorphism \(\gamma:\End(V)\to V\otimes V^*\). We can thus define the trace of \(T\in\End(V)\) as follows: let \(\gamma T = \sum_iv_i\otimes\alpha_i\in V\otimes V^*\). Then \[\tr\;T := \sum_i\alpha_i(v_i)\]
  15. Tensor product is adjoint to Hom: \[\Hom(U\otimes V, W)\simeq\Hom(U,\Hom(V, W)).\]

groups

(back to top)

  1. Groups are the simplest algebraic structures with enough complexity to be interesting.
  2. The set \(\{\id_G\}\) is a group, known as the trivial group.
  3. Every cyclic group is an abelian group.
  4. Subgroups of cyclic groups are cyclic.
  5. (Lagrange) The order of a subgroup \(H\) of a finite group \(G\) divides the order of \(G\).
  6. Modding out by a normal subgroup produces a quotient group.
  7. Every group homomorphism decomposes into a epimorphism followed by an isomorphism followed by a monomorphism.

rings

(back to top)

  1. A ring is an additive group in which one can multiply elements.
    • Every domain is a ring; every field is a domain.
    • One can form modules and algebras over a ring. A vector space is a module over a field.
  2. The set \(\{0\}\) is a ring, known as the zero ring.
  3. Rings have subrings and ideals. Modding out by an ideal produces a quotient ring.
  4. Domains are to prime ideals as fields are to maximal ideals.
  5. Every ring homomorphism decomoses into an epimorphism followed by an isomorphism followed by a monomorphism.
  6. The Chinese Remainder Theorem is the first local-global principle.
  7. Every domain has a set of localizations at each prime, the largest of which is its field of fractions.
  8. Closely related to prime ideals and maximal ideals are prime elements and irreducible elements.
  9. There are several types of rings between domains and fields.
    • Every field is a Euclidean domain.
    • Every Euclidean domain is a PID.
    • Every PID is a Noetherian domain in which primes are irreducible.
    • Every Noetherian domain in which primes are irreducible is a UFD.
    • Every UFD is an domain.
  10. Given a ring \(R\), one can form the polynomial ring over \(R\).

modules and algebras

(back to top)

  1. Modules are an abstract approach to linear algebra.
    • A vector space is a module over a field.
    • An abelian group is a module over the integers.
    • A module over the polynomial ring \(k[X]\) is the same thing as a vector space with a linear map.
  2. One can form submodules and quotient modules.
  3. \(\mathbf C\) as an \(\mathbf R\)-algebra is the archetypal algebra over a ring.

extensions of fields

(back to top)

  1. If \(\varphi:F\to E\) is a ring homomorphism where \(F\) is a field and \(E\) is a nonzero ring, then \(\varphi\) is injective.
  2. We tend to think of such homomorphisms as extensions, denoted \(E/F\), for example, or sometimes \(E\supset F\).
  3. Note that \(E\) is a vector space over \(F\). The dimension of this vector space is called the degree of the extension, denoted \([E:F]\).
  4. If \(K\supset E\supset F\) is a tower of fields, then \[[K:F] = [K:E][E:F].\]
  5. Automorphisms of an extension \(E/F\) form a group, denoted \(\Aut(E/F)\).
  6. There exists exactly one finite field up to isomorphism for each prime power.
