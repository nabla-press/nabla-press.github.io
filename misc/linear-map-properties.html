Here are some properties of linear maps. \(\def\id{\text{id}}\)

Let \(U, V, W\) be vector spaces over a given field \(\mathbf F\).

  1. If \(T:V\to W\) is a linear map, then \[T(0_V) = 0_W.\]
    [explantion] Let \(v\in V\). Then \[\begin{align*} T(0_V) &= T(v + (-v)) = Tv + T(-v)\\ &= Tv + T((-1)v) = Tv + (-1)Tv = Tv + (-Tv) = 0_W. \end{align*} \]
  2. If \(S:U\to V\) and \(T:V\to W\) are linear maps, then the composition \[TS:U\to W\] is also a linear map.
    [explantion] Let \(\alpha, \beta\in\mathbf F\) and \(u, v\in U\). Then \[ TS(\alpha u + \beta v) = T(\alpha Su + \beta Sv) = \alpha TS u + \beta TS v. \]
  3. There exists an identity map \(\id_U:U\to U\) given by \(u\mapsto u\). This map is linear.
    [explantion] Let \(\alpha, \beta\in\mathbf F\) and \(u, v\in U\). Then \[ \id_U(\alpha u + \beta v) = \alpha u + \beta v = \alpha\,\id_U u + \beta\,\id_Uv. \]
    Note that if \(S:U\to V\) is a linear map, then \[\id_VS = S = S\,\id_U.\]
  4. If \(T:V\to W\) and \(\widetilde T:V\to W\) are linear maps such that \[Tv = \widetilde Tv\] for all \(v\in V\), then \(T = \widetilde T\).
