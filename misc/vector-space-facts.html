The following are easy consequences of the vector space axioms:

  1. The origin 0 is unique.
    [explanation] Suppose \(0'\) is another origin. Then \(0 = 0 + 0' = 0'\).
  2. Every \(v\in V\) has a unique additive inverse (denoted \(-v\)).
    [explanation] Suppose \(w\) and \(w'\) are additive inverses of $v$. Then \[w = w + 0 = w + (v + w') = (w + v) + w' = 0 + w' = w'.\]
  3. The additive cancellation law holds: \(\forall u, v, w\in V\) we have \[u + w = v + w \implies u = v.\]
    [explanation] By direct computation: \[\begin{align*} u &= u + 0 = u + (w + (-w)) = (u + w) + (-w)\\ &= (v + w) + (-w) = v + (w + (-w)) = v + 0 = v. \end{align*} \]
  4. We have \(0v = 0\) for every \(v\in V\).
    [explanation] Computing, we find \[0 + 0v = 0v = (0 + 0)v = 0v + 0v\] so the result follows from the additive cancellation law.
  5. We have \(\alpha 0 = 0\) for every \(\alpha\in\mathbf F\).
    [explanation] Computing, we find \[0 + \alpha 0 = \alpha(0 + 0) = \alpha 0 + \alpha 0\] so the result follows from the additive cancellation law.
  6. We have \((-1)v = -v\) for every \(v\in V\).
    [explanation] By direct computation: \[\begin{align*} (-1)v &= (-1)v + 0 = (-1)v + (v + (-v))\\ &= (-1)v + (1v + (-v)) = ((-1)v + 1v) + (-v)\\ &= (-1 + 1)v + (-v) = 0v + (-v) = 0 + (-v) = -v \end{align*}\]
  7. We have \(\alpha v = 0\) implies either \(\alpha = 0\) or \(v = 0\).
    [explanation] Suppose \(\alpha v = 0\) and that \(\alpha\ne 0\). Then \[v = 1v = (\alpha^{-1}\alpha)v = \alpha^{-1}(\alpha v) = \alpha^{-1}0 = 0.\]
