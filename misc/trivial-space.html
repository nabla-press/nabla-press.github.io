The trivial space is the set \(\{0\}\) with the operation \(+(0, 0) = 0 + 0 = 0\) and action \((\lambda, 0)\mapsto\lambda 0 = 0\).

This forms a vector space.

[explanation] We check the axioms.
  1. Associativity of addition: \(0 + (0 + 0) = 0 + 0 = (0 + 0) + 0.\)
  2. Additive identity: \(0 + 0 = 0 = 0 + 0.\)
  3. Additive inverses: \(0 + 0 = 0 = 0 + 0.\)
  4. Commutativity of addition: \(0 + 0 = 0 + 0.\)
  5. Scalar identity: \((1)0 = 0\).
  6. Compatibility with scalar multiplication: \(\alpha(\beta 0) = \alpha 0 = 0 = (\alpha\beta)0\).
  7. Distributive law: \(\alpha(0 + 0) = 0 + 0 = \alpha 0 + \alpha 0\) and \((\alpha + \beta)0 = 0 = 0 + 0 = \alpha 0 + \beta 0\).
